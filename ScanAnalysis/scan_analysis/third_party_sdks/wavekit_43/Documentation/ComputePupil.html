static fit_legendre_pupil(pupil_in)[source]¶

Compute the geometric parameters of the Legendre pupil from a pupil input. This function is not configurable. It takes a user pupil as input and detects a square pupil that is as large as possible and entirely inside it. If several pupils with the same size suit, one of them is chosen arbitrarily.
