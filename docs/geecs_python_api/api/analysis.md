# Analysis Module (removed)

The `geecs_python_api.analysis` subpackage was removed in v0.4.0 as part of the unused-module purge. Data-loading utilities for GEECS scan data live in [Data Utils](../../geecs_data_utils/overview.md) instead — `ScanPaths`, `ScanData`, and the s-file loaders cover the use cases this module previously served.
